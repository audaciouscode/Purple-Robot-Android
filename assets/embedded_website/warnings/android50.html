Android 5.0 shipped with a serious memory leak that affects all applications on the system the longer the system runs:

  • Issue 79729: Memory Leak on Lollipop crashing Apps

  • Nasty Memory Leak in Android Lollipop's Rendering Engine is Killing Apps

  • Google Finally Fixes Android 5.0.1 Lollipop Memory Leak Issue

If you are using a device that has not yet received an Android 5.1 (or higher) update that addresses this issue, please restart your device daily to reclaim the leaked resources.

If you do not restart your device, all apps on the system will begin to crash more frequently. For Purple Robot, this will introduce gaps in data collection and prevent scheduled tasks from running on time.
