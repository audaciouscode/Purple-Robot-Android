Schedules a script to run at predefined time in the future. If a time in the past is specified, the script runs immediately.

Arguments

identifier - Unique string identifying the script. Subsequent calls to this function with the same identifier parameter will update any previously scheduled script with the same identifier.

dateString - String encoding the scheduled run time in the format yyyyMMdd'T'HHmmss. For example, noon on Christmas Day 2015 is encoded as 20151225T120000.

script - Script to run at the scheduled time.

Return Values

None

JavaScript Example

PurpleRobot.scheduleScript("my-script", "20151225T120000", "PurpleRobot.playDefaultTone();");

Scheme Example

(pr-schedule-script "my-script" "20151225T120000" "(pr-play-default-tone)")
