Launches a local Android app with human-readable name or package name matching applicationName.

Arguments

applicationName - Name of the specified app.

options (Optional) - Dictionary or dictionary-like object containing key-value pairs to pass to the application's launch Intent.

script (Optional) - Script to run after the successful launch of the specified app.

Return Values

Boolean true if the app was found and launched, false otherwise.

JavaScript Example

PurpleRobot.launchApplication("Gmail");

PurpleRobot.launchApplication("Maps", { "foo": "bar" }, "PurpleRobot.playDefaultTone()");

Scheme Example

(pr-launch-application "Gmail")

(pr-launch-application "Maps" '(("foo" . "bar")) "(pr-play-default-tone)")
