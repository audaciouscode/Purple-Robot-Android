Displays a notification in the status bar that runs script when tapped.

See PurpleRobot.cancelScriptNotification for a mechanism for dismissing these notifications.

Arguments

title - Notification title string.

message - Notification message string.

persistent - Boolean specifying if the notification must be tapped to be dismissed (no swipe to clear).

script - Purple Robot script to run when a user taps the notification.

sticky (optional) - Boolean specifying if the notification should not disappear when tapped.

imageUrl - URL of the image to display as part of the notification.

Return Values

None

JavaScript Example

PurpleRobot.showScriptNotification("My Notification", "Tap here for magic!", true, "PurpleRobot.playDefaultTone();", true, "http://farm1.staticflickr.com/22/33282399_4285868130.jpg");

Scheme Example

(pr-show-script-notification "My Notification" "Tap here for magic!" #t "(pr-play-default-tone)" #t "http://farm1.staticflickr.com/22/33282399_4285868130.jpg")
