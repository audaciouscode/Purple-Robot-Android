Plays the default audio notification tone on the device.

Arguments

loops (Optional) - Boolean value specifying whether to repeat playing the tone in a loop. Default is false.

See PurpleRobot.stopPlayback for details about canceling a looping tone.

Return Values

None

JavaScript Example

PurpleRobot.playDefaultTone();

PurpleRobot.playDefaultTone(true);

Scheme Example

(pr-play-default-tone)

(pr-play-default-tone #t)
