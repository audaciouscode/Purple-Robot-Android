Retrieves the value saved to the settings using the namespace namespaceId.

Arguments

namespaceId - Namespace to retrieve.

Return Values

Dictionary or dictionary-like object containing the values in the specified namespace.

JavaScript Example

PurpleRobot.persistString("my-new-namespace", "FOO", "BAR");

PurpleRobot.fetchNamespace("my-new-namespace");

Scheme Example

(pr-persist-string "my-new-namespace" "FOO" "BAR")

(pr-fetch-namespace "my-new-namespace")
