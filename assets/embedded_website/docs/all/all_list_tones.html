Retrieves a list of notification tones on the device.

Arguments

None

Return Values

A list of the notification tones available on the local devices, suitable for use in PurpleRobot.playTone.

JavaScript Example

var tones = PurpleRobot.listTones();

PurpleRobot.playTone(tones[0]);

Scheme Example

(let ((tones (pr-list-tones)))
     (pr-play-tone (pr-nth 0 tones)))
