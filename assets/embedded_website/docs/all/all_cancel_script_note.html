Dismisses a notification created with PurpleRobot.showScriptNotification.

Arguments

None

Return Values

None

JavaScript Example

PurpleRobot.showScriptNotification("My Notification", "Tap here for magic!", true, "PurpleRobot.playDefaultTone();", true);
PurpleRobot.cancelScriptNotification();

Scheme Example

(pr-show-script-notification "My Notification" "Tap here for magic!" #t "(pr-play-default-tone)" #t)
(pr-cancel-script-notification)
