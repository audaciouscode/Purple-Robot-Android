Plays the audio notification specified by tone on the device. If the specified notification cannot be found, the default notification tone is played instead.

Arguments

tone - Name of the notification tone to play. See PurpleRobot.listTones for available options.

loops (Optional) - Boolean value specifying whether to repeat playing the tone in a loop. Default is false.

See PurpleRobot.stopPlayback for details about canceling a looping tone.

Return Values

None

Discussion

The names and availability of specific tones will vary by device and manufacturer.

Purple Robot Tones

  • CRcv
  • Csnd
  • Error
  • New
  • Off
  • On

These tones originated as part of the Tokyo Train Station Adium plugin.

JavaScript Example

PurpleRobot.playTone('CRcv');

PurpleRobot.playTone('CSnd', true);

Scheme Example

(pr-play-tone "CRcv")

(pr-play-tone "CSnd" #t)
