Broadcasts an intent to other apps on the device.

Arguments

action - Intent action.

extras - Dictionary of extras to include with the intent

Return Values

Boolean value indicating the success of the broadcast.

JavaScript Example

PurpleRobot.broadcastIntent("my-custom-action", { "foo": "bar" });

Scheme Example

(pr-broadcast-intent "my-custom-action" '(("foo" . "bar")))
