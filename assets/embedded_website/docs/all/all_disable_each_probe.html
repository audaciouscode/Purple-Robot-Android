Disables every single Purple Robot probe. This is equivalent to calling PurpleRobot.disableProbe for every available probe. It is not equivalent to calling PurpleRobot.disableProbes, which does not alter the individual probes' configuration.

Arguments

None

Return Values

None

JavaScript Example

PurpleRobot.disableEachProbe();

Scheme Example

(pr-disable-each-probe)
