Launches a local Android activity configured to open url. Note that not all URLs point to web pages this function can open app-specific locations.

Arguments

url - URL to open locally.

Return Values

Boolean true if the app was found to launch url, false otherwise.

JavaScript Example

PurpleRobot.launchUrl("http://cbits.northwestern.edu");

PurpleRobot.launchUrl("geo:0,0?q=1600+Amphitheatre+Parkway%2C+CA");

Scheme Example

(pr-launch-url "http://cbits.northwestern.edu")

(pr-launch-url "geo:0,0?q=1600+Amphitheatre+Parkway%2C+CA")
