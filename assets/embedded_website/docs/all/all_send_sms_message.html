Sends an SMS message.

Arguments

message: String containing message to transmit.

destination: Phone destination for the message. An empty string or null will send the message to the default SMS number.

Return Values

None

JavaScript Example

PurpleRobot.sendSmsMessage("Hello world!", "8675309");
PurpleRobot.sendSmsMessage("Hello world!");

Scheme Example

(pr-send-sms-message "Hello World!" "8675309")
(pr-send-sms-message "Hello World!")
