Launches an an interface to scan NFC tagson the local device. Returns true if the device can read NFC tags, false if the device lacks the necessary hardware and software.

See PurpleRobot.canScanNFC for details about testing for NFC features, and PurpleRobot.cancelNFCScan to dismiss an active scan.

Arguments

None

Return Values

Boolean value indicating whether the system is equipped with NFC hardware.

JavaScript Example

PurpleRobot.scanNFC();

Scheme Example

(pr-scan-nfc)
