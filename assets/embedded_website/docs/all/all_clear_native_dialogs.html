Removes from the screen and the display queue all dialogs previously created by PurpleRobot.showNativeDialog with tag.

If the tag parameter is omitted all dialogs are cleared.

Arguments

tag (optional)- Identifier for one or a group of dialogs. If omitted, Purple Robot clears all dialogs.

Return Values

None

JavaScript Example

PurpleRobot.showNativeDialog("My Native Dialog", "This is my native dialog.\n\nWhat do you think about it?", "Groovy", "Meh", "PurpleRobot.emitToast(\"Thanks!\");", "PurpleRobot.emitToast(\"We'll try harder next time.\");", "my-tag", 1);
PurpleRobot.clearNativeDialogs("my-tag");

PurpleRobot.showNativeDialog("Another Dialog", "This is another dialog.\n\nWhat do you think about it?", "Groovy", "Meh", "PurpleRobot.emitToast(\"Thanks!\");", "PurpleRobot.emitToast(\"We'll try harder next time.\");", "another-tag", 2);
PurpleRobot.clearNativeDialogs();

Scheme Example

(pr-show-native-dialog "My Native Dialog" "This is my native dialog.\n\nWhat do you think about it?" "Groovy" "Meh" "PurpleRobot.emitToast(\"Thanks!\");" "PurpleRobot.emitToast(\"We'll try harder next time.\");" "my-tag" 1)
(pr-clear-native-dialogs "my-tag")

(pr-show-native-dialog "Another Dialog" "This is another dialog.\n\nWhat do you think about it?" "Groovy" "Meh" "PurpleRobot.emitToast(\"Thanks!\");" "PurpleRobot.emitToast(\"We'll try harder next time.\");" "another-tag" 0)
(pr-clear-native-dialogs)
