Retrieves the details for the snapshot with identifier snapshotId.

Arguments

snapshotId: Unique identifier for the desired snapshot.

Return Values

Dictionary or dictionary-like object containing details and probe readings at the time the snapshot was taken.

JavaScript Example

PurpleRobot.takeSnapshot("test-context");

PurpleRobot.fetchSnapshot(PurpleRobot.fetchSnapshotIds()[0]);

Scheme Example

(begin (pr-take-snapshot "test-context")
       (pr-fetch-snapshot (first (pr-fetch-snapshot-ids))))
