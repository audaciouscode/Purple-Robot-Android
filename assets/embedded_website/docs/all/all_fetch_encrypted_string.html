Retrieves and decodes the encrypted value saved to the local persistence store using key previously persisted with PurpleRobot.persistEncryptedString.

If collections of values should be aggregated into separate collections - to support multiple simultaneous Purple Robot clients - the scripting API provides an alternative function where a namespace may be specified:

PurpleRobot.fetchEncryptedString(namespace, key)

Values saved with a namespace will not be included in the output of the PurpleRobot.fetchNamespace function.

See PurpleRobot.persistEncryptedString for more details about the encryption and keys used to encode the persisted value.

Arguments

key - Key used to retrieve previously persisted value.

namespace (Optional) - Namespace containing the key/value pair.

Return Values

String containing the previously persisted value, null if entry exists under key.

JavaScript Example

PurpleRobot.fetchEncryptedString("foo");

PurpleRobot.fetchEncryptedString("my-namespace", "foo",);

Scheme Example

(pr-fetch-encrypted-string "foo")

(pr-fetch-encrypted-string "my-namespace" "foo")
