Saves value to the local persistence store. value can be retrieved by calling PurpleRobot.fetchString with key.

If collections of values should be aggregated into separate collections - to support multiple simultaneous Purple Robot clients - the scripting API provides an alternative function where a namespace may be specified:

PurpleRobot.persistString(namespace, key, value)

values saved with a namespace parameter may be retrieved with other key/value pairs persisted in the same namespace using the PurpleRobot.fetchNamespace function.

Arguments

key - Key used for later retrieval.

value - String to be persisted.

namespace (Optional) - Namespace to contain the key/value pair.

Return Values

Boolean value indicating whether value was persisted successfully.

JavaScript Example

PurpleRobot.persistString("foo", "bar");

PurpleRobot.persistString("my-namespace", "foo", "bar");

Scheme Example

(pr-persist-string "foo" "bar")
