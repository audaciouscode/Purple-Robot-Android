Displays a Purple Robot native dialog with the provided values.

confirmLabel and cancelLabel provide the label for the dialog's buttons. If either of those values are empty strings, the corresponding buttons will be omitted from the dialog.

confirmScript and cancelScript are variables containing a Purple Robot script that will be run when the corresponding button is pressed. If an empty script is provided, the corresponding button press will only dismiss the dialog and not execute any further actions.

The optional string parameter tag is used to group related dialogs. This is primarily used in the PurpleRobot.clearNativeDialogs function where all dialogs with the same tag will be dismissed and removed from the display queue when that function is invoked with a matching tag parameter.

The order of dialogs can be manipulated by using the optional priority parameter. Priorities are specified as integers and dialogs with higher priorities will take display precendece over lower priority dialogs. Dialogs dismissed because of prioritization will reappear immediately after the higher priority dialogs close.

Arguments

title - Dialog title.

message - Dialog message.

confirmLabel - Label for the positive action button. May be an empty string to omit the positive button.

cancelLabel - Label for the negative action button. May be an empty string to omit the negative button.

confirmScript - Script to be executed when the positive action button is pressed. May be an empty string for no action.

cancelScript - Script to be executed when the negative action button is pressed. May be an empty string for no action.

tag (optional) - String associating the dialog to a group of dialogs with the same tag.

priority (optional) - Disply priority of the dialog. Higher priority dialogs will appear above and before lower priority dialogs.

Return Values

None

JavaScript Example

PurpleRobot.showNativeDialog("My Native Dialog", "This is my native dialog.\n\nWhat do you think about it?", "Groovy", "Meh", "PurpleRobot.emitToast(\"Thanks!\");", "PurpleRobot.emitToast(\"We'll try harder next time.\");");

PurpleRobot.showNativeDialog("Low Priority Dialog", "I'm not so important.", "Ok", "", "", "", "low-priority", 1);

PurpleRobot.showNativeDialog("High Priority Dialog", "I'm super important!", "Ok", "", "", "", "high-priority", 10);
