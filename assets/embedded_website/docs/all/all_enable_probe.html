Enables a single Purple Robot probe named probeName.

Note that if probe data collection is disabled (via the settings or PurpleRobot.disableProbes, calling this function will not automatically reenable probe data collection. Call PurpleRobot.enableProbes to enable probe data collection.

Arguments

probeName: Name of the probe to enable. See the probe documentation for a full list of probes and their names.

Return Values

None

JavaScript Example

PurpleRobot.enableProbe("edu.northwestern.cbits.purple_robot_manager.probes.builtin.RobotHealthProbe");

Scheme Example

(pr-enable-probe "edu.northwestern.cbits.purple_robot_manager.probes.builtin.RobotHealthProbe")
