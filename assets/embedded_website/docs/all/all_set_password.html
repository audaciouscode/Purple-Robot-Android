Sets the configuration password for the Purple Robot instance.

If a configuration password is set, the end user will be unable to access the Purple Robot settings without the specified password. This is often used to lock configurations for studies in the field.

See PurpleRobot.clearPassword for a function to reset the configuration password.

Arguments

password: String containing the new password. An empty string or null will remove the configuration password and allow unfettered access to the Purple Robot settings.

Return Values

None

JavaScript Example

PurpleRobot.setPassword("change-me");

Scheme Example

(pr-set-password "change-me")
