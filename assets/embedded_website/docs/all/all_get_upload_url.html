Returns upload endpoint URL where Purple Robot is configured to transmit sensor data and other readings.

See PurpleRobot.setUploadUrl for details to set the upload endpoint URL.

Arguments

None

Return Values

String containing the upload endpoint URL, null if no upload URL has yet been configured.

JavaScript Example

PurpleRobot.getUploadUrl();
