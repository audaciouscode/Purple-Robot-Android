Deactivates passive data collection using Purple Robot probes.

Note that this function disables overall probe data collection, it does not disable any individual probe. Thus, any probes that were previously enabled before calling this function would resume data collection when PurpleRobot.enableProbes is called. Any probes that were disabled would remain disabled.

To toggle the state of any individual probe, see PurpleRobot.enableProbe and PurpleRobot.disableProbe. To disable each individual probe PurpleRobot.disableEachProbe.

Arguments

None

Return Values

None

JavaScript Example

PurpleRobot.disableProbes();

Scheme Example

(pr-disable-probes)
