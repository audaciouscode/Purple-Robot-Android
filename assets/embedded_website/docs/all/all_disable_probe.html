Disables a single Purple Robot probe named probeName.

Arguments

probeName: Name of the probe to disable. See the probe documentation for a full list of probes and their names.

Return Values

None

JavaScript Example

PurpleRobot.disableProbe("edu.northwestern.cbits.purple_robot_manager.probes.builtin.RobotHealthProbe");

Scheme Example

(pr-disable-probe "edu.northwestern.cbits.purple_robot_manager.probes.builtin.RobotHealthProbe")
