Batch updates a variety of setting values of the Purple Robot instance.

Note that a version of this function exists to update a single setting at once: PurpleRobot.updateConfig(key, value).

Arguments

configMap - Dictionary or dictionary-like structure containing the setting names and values to apply to the Purple Robot instance.

Return Values

Boolean value indicating whether the settings were updated successfully.

JavaScript Example

PurpleRobot.updateConfig({ 
    "config_mute_warnings": true, 
    "config_builtin_http_server_password": "cant-guess-this",
    "config_log_upload_interval": "60000"
});

Scheme Example

(pr-update-config '(("config_mute_warnings" . #t)
                    ("config_builtin_http_server_password" . "cant-guess-this")
                    ("config_log_upload_interval" . "60000")))
