Loads an included JavaScript library into the current execution context.

Arguments

library - Library name. Included with Purple Robot:

  • astro.js
  • date.js
  • jstat.js
  • moment.js
  • underscore.js

Return Values

Any results returned from script.

JavaScript Example

PurpleRobot.loadLibrary("moment.js");

moment().format('MMMM Do YYYY, h:mm:ss a');
