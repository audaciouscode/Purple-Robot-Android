Runs script using the local JavaScript engine.

Arguments

script - JavaScript program to run.

extras (Optional) - Name of the custom object to inject into the JavaScript runtime.

extraValues (Optional) - Values for the custom object to inject into the JavaScript runtime.

Return Values

Boolean true if the library was imported successfully, false otherwise.

JavaScript Example

PurpleRobot.runScript("PurpleRobot.playDefaultTone();");

PurpleRobot.runScript("PurpleRobot.vibrate(pattern['name']);", "pattern", { "name": "Buzz" });
