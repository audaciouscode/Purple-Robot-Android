Schedules a script to run at after the specified delay. If a a negative delay is specified, the script runs immediately.

Arguments

identifier - Unique string identifying the script. Subsequent calls to this function with the same identifier parameter will update any previously scheduled script with the same identifier.

delay - Number of milliseconds to delay before running the script.

script - Script to run after the specified delay.

Return Values

None

JavaScript Example

PurpleRobot.scheduleScriptDelayed("my-script", 5000, "PurpleRobot.playDefaultTone();");

Scheme Example

(pr-schedule-script-delayed "my-script" 5000 "(pr-play-default-tone)")
