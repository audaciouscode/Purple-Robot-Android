Activates passive data collection using Purple Robot probes.

Note that this function enables overall probe data collection, it does not enable any individual probe. Thus, any probes that were previously enabled before calling PurpleRobot.disableProbes would resume data collection. Any probes that were disabled would remain disabled.

To toggle the state of any individual probe, see PurpleRobot.enableProbe and PurpleRobot.disableProbe.

Arguments

None

Return Values

None

JavaScript Example

PurpleRobot.enableProbes();

Scheme Example

(pr-enable-probes)
