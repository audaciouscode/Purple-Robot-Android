Returns true when the system has an enabled DataUploadPlugin that is persisting probe readings, false if no sensor and other emitted readings are not being saved for later transmission.

Arguments

None

Return Values

Boolean value indicating whether the system has an enabled DataUploadPlugin that is currently saving sensor readings and other emitted data for transmission off-device.

JavaScript Example

PurpleRobot.uploadersEnabled();
