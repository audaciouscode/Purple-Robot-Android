Launches url in a local WebView within Purple Robot.

Note that this only supports URLs that would be opened in a local web browser.

Arguments

url - URL to open locally.

Return Values

Boolean true if url was opened successfully, false otherwise.

JavaScript Example

PurpleRobot.launchInternalUrl("http://cbits.northwestern.edu");

Scheme Example

(pr-launch-internal-url "http://cbits.northwestern.edu")
