Displays a Purple Robot notification in the Android system bar that will launch the app with the package packageName when tapped.

Arguments

title - Notification title.

message - Notification message.

packageName - Package name of the application to launch on tap.

Return Values

True if the notification was scheduled or fired successfully, false if an error occurred.

JavaScript Example

PurpleRobot.showApplicationLaunchNotification("Tap here to launch Facebook.", "If you tap this notification, Facebook will open on your device (if installed).", "com.facebook.katana");

Scheme Example

(pr-show-application-launch-notification "Tap here to launch Facebook." "If you tap this notification, Facebook will open on your device (if installed)." "com.facebook.katana")
