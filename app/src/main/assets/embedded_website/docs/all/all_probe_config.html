Retrieves a probe's configuration.

See the probe documentation for probe names.

Arguments

probeName - Name of the probe with configuration to retrieve.

Return Values

Dictionary or dictionary-like object containing the probe's current configuration.

JavaScript Example

PurpleRobot.probeConfig("edu.northwestern.cbits.purple_robot_manager.probes.builtin.RobotHealthProbe");

Note that due to an issue with the underlying Rhino JavaScript library, Boolean configuration values are omitted. To determine the values (and existence) of these parameters, please use the Scheme scripting API instead.

Scheme Example

(pr-probe-config "edu.northwestern.cbits.purple_robot_manager.probes.builtin.RobotHealthProbe")
