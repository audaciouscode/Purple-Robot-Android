Displays a toast notification containing the string in message.

By default, the toast is visible only for a short period of time, unless a true Boolean argument is provided to specify a longer duration.

Arguments

message - String to appear in the toast notification.

useLongDuration (Optional) - True if the toast should display for a longer period of time, false otherwise.

Return Values

None

JavaScript Example

PurpleRobot.emitToast("This is a short message...");

PurpleRobot.emitToast("This is also a short message...", false);

PurpleRobot.emitToast("This will be visible longer...", true);

Scheme Example

(pr-emit-toast "This is a short message...")

(pr-emit-toast "This is also a short message..." #f)

(pr-emit-toast "This will be visible longer..." #t)
