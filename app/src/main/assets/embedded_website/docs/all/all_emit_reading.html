Generates a synthetic probe reading and transmits it through the Purple Robot data processing facilities.

Arguments

name - String identifying the reading.

value - Value of the reading. value may consist of a string, number, or dictionary-like object that can be expressed with JSON-compatible or Scheme-compatible types.

priority (Optional) - Boolean specifying if this is a high-priority reading. For example, readings with high-priorities will be transmitted earlier by some DataUploadPlugins than readings with no priority specified.

Return Values

None

JavaScript Example

PurpleRobot.emitReading("myFavoriteNumber", 7);

PurpleRobot.emitReading("myFavoriteColor", "blue", true);

PurpleRobot.emitReading("myFavoriteFoods", [ "pizza", "pancakes", "cheese" ]);

PurpleRobot.emitReading("myFavoriteCar", { "mfgr": "Ford", "model": "Mustang", "engine": "5.0L V8" }, true);

Scheme Example

(pr-emit-reading "myFavoriteNumber" 7)

(pr-emit-reading "myFavoriteColor" "blue" #t)

(pr-emit-reading "myFavoriteCar" '(("mfgr" . "Ford")
                                   ("model" . "Mustang")
                                   ("engine" . "5.0L V8")) #t)
