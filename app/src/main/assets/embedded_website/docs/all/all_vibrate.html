Vibrates the device using the sequence named pattern.

pattern may be one several values:

  • Blip: A single short vibration.
  • Buzz: A single long vibration.
  • S.O.S.: The international Morse code distress signal.

In addition to pattern, an optional Boolean repeats may also be provided to specify if the pattern should loop repeatedly until PurpleRobot.stopVibrate is called.

Arguments

pattern - Name of the vibration pattern to use.

repeats - True if the pattern should repeat indefinately, false otherwise. (optional)

Return Values

None

JavaScript Example

PurpleRobot.vibrate("Buzz");

PurpleRobot.vibrate("Blip", false);

PurpleRobot.vibrate("S.O.S.", true);

Scheme Example

(pr-vibrate "Buzz")

(pr-vibrate "Blip" #f)

(pr-vibrate "S.O.S." #t)
