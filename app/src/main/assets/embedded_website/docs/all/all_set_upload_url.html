Configures Purple Robot to send sensor data and other readings to the upload endpoint located at url.

See PurpleRobot.getUploadUrl for details to retrieve the currently configured upload endpoint URL.

Arguments

url - Upload endpoint URL implementing the Purple Robot data upload API.

Return Values

None

JavaScript Example

PurpleRobot.setUploadUrl("http://www.example.com/pr/");
