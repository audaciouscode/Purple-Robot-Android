Returns true if the local device is equipped with an NFC scanning initiator, false if the system cannot read NFC tags.

See PurpleRobot.scanNFC for details about initiating an NFC scan, and PurpleRobot.cancelNFCScan to dismiss an active scan.

Arguments

None

Return Values

Boolean value indicating whether the system is equipped with NFC hardware.

JavaScript Example

PurpleRobot.canScanNFC();

Scheme Example

(pr-can-scan-nfc?)
