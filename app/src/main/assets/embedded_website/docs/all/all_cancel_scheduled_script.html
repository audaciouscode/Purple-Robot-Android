Cancels a script with identifier previously scheduled by PurpleRobot.scheduleScript.

Arguments

identifier - Unique identifier of the script to remove.

Return Values

None

JavaScript Example

PurpleRobot.scheduleScript("my-script", "20151225T120000", "PurpleRobot.playDefaultTone();");
PurpleRobot.scheduledScripts();

PurpleRobot.cancelScheduledScript("my-script");
PurpleRobot.scheduledScripts();

Scheme Example

(pr-schedule-script "my-script" "20151225T120000" "(pr-play-default-tone)")
(pr-scheduled-scripts)

(pr-cancel-scheduled-script "my-script")
(pr-scheduled-scripts)
