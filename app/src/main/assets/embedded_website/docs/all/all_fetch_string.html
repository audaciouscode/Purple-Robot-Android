Retrieves the value saved to the local persistence store using key previously persisted with PurpleRobot.persistString.

If collections of values should be aggregated into separate collections - to support multiple simultaneous Purple Robot clients - the scripting API provides an alternative function where a namespace may be specified:

PurpleRobot.fetchString(namespace, key)

Values saved with a namespace parameter may be retrieved with other key/value pairs persisted in the same namespace using the PurpleRobot.fetchNamespace function.

Arguments

key - Key used to retrieve previously persisted value.

namespace (Optional) - Namespace containing the key/value pair.

Return Values

String containing the previously persisted value, null if entry exists under key.

JavaScript Example

PurpleRobot.fetchString("foo");

PurpleRobot.fetchString("my-namespace", "foo",);

Scheme Example

(pr-fetch-string "foo")

(pr-fetch-string "namespace" "foo").
