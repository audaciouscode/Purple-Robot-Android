Returns the string content located at url. An optional lenient parameter may be passed to relax SSL verification requirements for servers with self-signed certificates. (By default, full verification is enabled.)

Arguments

url - URL containing the string content to retrieve.

lenient (Optional) - Boolean value specifying whether strict SSL checks should be bypassed.

Return Values

String containing the content of url, null if binary content is retrieved or an error occurs.

JavaScript Example

PurpleRobot.readUrl("http://www.example.com/");

PurpleRobot.readUrl("https://example.com", true);

Scheme Example

(pr-read-url "http://www.example.com/")

(pr-read-url "https://www.example.com/" #t)
