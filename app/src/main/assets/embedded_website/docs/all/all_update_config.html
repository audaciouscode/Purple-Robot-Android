Updates a single setting value of the Purple Robot instance.

Note that a batch version of this function exists to update multiple settings at once: PurpleRobot.updateConfig(mapObject).

Arguments

key - Key used by the internal SharedPreferences object.

value - String or Boolean value of the updated setting. Note that numeric values (such as upload intervals) are persisted internally as strings to work with the Android settings UI infrastructure.

Return Values

Boolean value indicating whether the setting was updated successfully.

JavaScript Example

// Mute sanity check warnings...
PurpleRobot.updateConfig("config_mute_warnings", true);

// Set a password on the embedded HTTP server...
PurpleRobot.updateConfig("config_builtin_http_server_password", "cant-guess-this");

// Configure the logging system to attempt an upload once every 60 seconds...
PurpleRobot.updateConfig("config_log_upload_interval", "60000");

Scheme Example

;;; Mute sanity check warnings...
(pr-update-config "config_mute_warnings" #t)

;;; Set a password on the embedded HTTP server...
(pr-update-config "config_builtin_http_server_password" "cant-guess-this")

;;; Configure the logging system to attempt an upload once every 60 seconds...
(pr-update-config "config_log_upload_interval" "60000")
