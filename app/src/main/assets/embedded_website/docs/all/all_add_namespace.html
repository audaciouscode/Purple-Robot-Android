Registers a new empty namespace for value persistence if a namespace identified by namespace does not already exist.

If a namespace does not already exist when calling PurpleRobot.persistString, a new one will automatically be registered, thus this method is only necessary for creating empty namespaces.

Arguments

namespace: Identifier of the new namespace.

Return Values

None

JavaScript Example

PurpleRobot.addNamespace('my-new-namespace');

PurpleRobot.namespaces();

Scheme Example

(pr-add-namespace "my-new-namespace")

(pr-namespaces)
