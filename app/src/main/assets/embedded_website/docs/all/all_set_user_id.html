Sets the user identifier for the Purple Robot instance.

User identifiers are primarily used to determine the ownership of transmitted data. Uploaded payloads are tagged with a one-way hash of the user identifer to specify the source of the data. See PurpleRobot.fetchUserHash for more specifics about the hash generation.

Arguments

userId: String containing the new user identifier.

refresh (Optional): Boolean value specifying whether the device configuration should be refreshed with the new identifier. Default is true.

Return Values

None

JavaScript Example

PurpleRobot.setUserId("foo@bar.com");

PurpleRobot.setUserId("foo@bar.com", false);

Scheme Example

(pr-set-user-id "foo@bar.com")

(pr-set-user-id "foo@bar.com" #f)
