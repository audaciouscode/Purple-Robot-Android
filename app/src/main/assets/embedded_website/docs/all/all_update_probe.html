Updates a probe's configuration.

See the probe documentation for probe names and configuration parameters.

Arguments

parameters - Dictionary or dictionary-like structure containing the configuration names and values to apply to the specified probe.

Return Values

Boolean value indicating whether the probe was updated successfully.

JavaScript Example

PurpleRobot.probeConfig("edu.northwestern.cbits.purple_robot_manager.probes.builtin.RobotHealthProbe");

PurpleRobot.updateProbe({
  "name": "edu.northwestern.cbits.purple_robot_manager.probes.builtin.RobotHealthProbe",
  "frequency": 300000
});

Scheme Example

(pr-probe-config "edu.northwestern.cbits.purple_robot_manager.probes.builtin.RobotHealthProbe")

(pr-update-probe '(("name" . "edu.northwestern.cbits.purple_robot_manager.probes.builtin.RobotHealthProbe")
                   ("frequency" . 300000L)))
